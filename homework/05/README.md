# 購物車網站專案

三個版本的購物車網站，逐步從靜態 HTML 進化到完整互動購物車。

## 版本說明

### v1 — 純 HTML

僅使用 HTML 建立商品列表與購物車區塊，無任何樣式與互動。

**檔案：** `v1/index.html`

### v2 — HTML + CSS

加入 CSS 樣式美化，使用 Flexbox/Grid 排版、卡片式設計、hover 動畫效果。

**檔案：** `v2/index.html`, `v2/style.css`

### v3 — HTML + CSS + JavaScript

完整購物車功能：

- 動態渲染商品列表
- 加入 / 移除購物車
- 數量增減（+ / −）
- 自動計算總金額
- 購物車數量徽章
- 清空購物車 / 結帳

**檔案：** `v3/index.html`, `v3/style.css`, `v3/script.js`

## 使用方式

直接用瀏覽器開啟各版本的 `index.html` 即可檢視。

v3 可直接在網頁上操作購物車所有功能。
